### 特殊的几个矩阵或关系
**零矩阵**：如果一个矩阵的所有元素都是0。

**同型矩阵**：A和B都是m*n的矩阵，就称它们为同型矩阵。


### 常用运算公式

"组合":  
- $(A^{T})^{-1} = (A^{-1})^{T}$
- $(A^{*})^{-1} = (A^{-1})^{*} = \frac{1}{|A|}A$
- $(A^{*})^{T} = (A^{T})^{*}$

"自组合"：
- $(A^{T})^{T} = A$
- <rm>$(A^{*})^{*} = |A|^{n-2}A \ (n \ge 2)$</rm>
- $(A^{-1})^{-1} = A$

"数乘"：
- $(kA)^{T} = kA^{T}$
- $(kA)^{*} = k^{n-1}A^{*}$
- $(kA)^{-1} = \frac{1}{k}A^{-1}$

"相加"：
- $(A + B)^{T} = A^{T} + B^{T}$
- $(A + B)^{*} 不确定$
- $(A + B)^{-1} 不确定$

"相乘"
- $(AB)^{T} = B^{T}A^{T}$
- $(AB)^{*} = B^{*}A^{*}$
- $(AB)^{-1} = B^{-1}A^{-1}$

"行列式"
- $|A^{T}| = |A|$
- $|A^{*}| = |A|^{n-1}$
- $|A^{-1}| = \frac{1}{|A|}$
